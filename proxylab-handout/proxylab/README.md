# Proxy Lab

## 项目概述
该项目实现了一个简单的代理服务器，能够处理客户端请求并将其转发到目标服务器，同时接收响应并将其返回给客户端。代理服务器还实现了缓存功能，以提高性能和减少对目标服务器的请求。

## 文件说明
- **proxy.c**: 代理服务器的主要实现，负责处理客户端请求、转发请求到目标服务器、接收响应并将其返回给客户端，同时管理缓存的使用。
- **cache.c**: 实现缓存功能，包括存储、检索和管理缓存对象的逻辑，提供对缓存的操作接口。
- **cache.h**: 缓存模块的头文件，声明缓存相关的结构体和函数接口，以供其他文件使用。
- **csapp.c**: 实现一些通用的辅助函数，通常用于网络编程和系统调用，提供对套接字、文件描述符等的封装。
- **csapp.h**: `csapp.c` 的头文件，声明在 `csapp.c` 中实现的函数和数据结构。
- **Makefile**: 用于构建项目，定义编译器、编译选项和目标文件的依赖关系，包含编译和清理的规则。
- **README.md**: 项目的文档，说明项目的目的、使用方法和其他相关信息。

## 使用说明
1. **编译项目**: 在项目根目录下运行 `make` 命令以编译项目。
2. **运行代理服务器**: 使用编译生成的可执行文件启动代理服务器。
3. **配置客户端**: 将客户端的代理设置指向运行中的代理服务器。

## 其他信息
- 项目遵循 C 语言编程规范，使用了多线程和缓存机制以提高性能。
- 该项目适用于学习网络编程和代理服务器的基本原理。
